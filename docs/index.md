# SnapPDF

A zero-dependency, feature-rich HTML+JS web app that converts photos into a printable A4 PDF with waterfall grid layout, drag-and-drop sorting, per-photo rotation, zoom, page navigator, live preview, and drag-to-reorder page groups.

## Getting Started

Just open `index.html` in any modern browser. No install, no build step, no backend.

## Features

- Waterfall grid layout (1-10 columns)
- Portrait & Landscape A4
- Drag-and-drop reordering
- Per-photo rotation (CW/CCW)
- Zoom slider (50-200mm)
- Page Navigator with click-to-jump
- Live A4 preview with pagination
- Page group reordering
- Custom filename & JPEG quality
- Auto EXIF rotation
- Mobile responsive
