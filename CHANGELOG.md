## [0.1.0] — 2026-06-20

### Added
- Initial project structure
- Frontend-only SnapPDF web app (index.html, 1779 lines)
- Waterfall multi-column photo layout (1–10 columns)
- A4 Portrait & Landscape support
- Drag & drop photo sorting (SortableJS)
- EXIF-aware rotation (manual CW/CCW)
- Custom captions per photo
- JPEG quality control
- Multi-page PDF generation (jsPDF)
- Responsive dark/light theme
- Core module scaffold
- CI/CD workflow with GitHub Actions
